A* Path Finder

The A* Pathfinder is a program designed for visualizing the A* algorithm. The program displays a grid of visitable nodes which are color coded based on algorithm progress. Additionally the program is slowed to allow the user to see what the algorithm is doing.

Phillips Hue Light Controller

View code

A smarthome type program designed to run on the raspberry pi and official raspberry pi display. The program displays temperature, weather, date, time, and allows controlling of lights on the connected Phillips Hue Bridge. Additionally the program can display computer temperature and load data live from another pc. Built with Java and JavaFX.
